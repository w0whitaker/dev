My journey as a developer has gone a little something like this:

"how do I do such and such a thing?" -> google it -> "oh, I need to use this tool" -> google it -> "ok, well that didn't work" -> google it -> "now I'm just confused" -> stack overflow -> "I know nothing" -> find a tutorial that answers some questions but raises others-> "ok, this time I'm going to start at the beginning and really learn this tool" -> the months pass -> "what was it I was trying to do?"

Thus it was with Vim. I don't mean to disparage Vim, or to discount its value - in many ways it is a joy to use, and once you've climbed that hill, it's hard not to miss it. I also don't mean to single Vim out; really, at least in my case, you could substitute any number of tools (Ansible, Vagrant, etc.) and the feeling would be the same.

But, fuck Vim. Srsly.

If I could have back all the time I spent learning how to use Vim and set up various development environments, I'd learn HTML backwards and forwards. Ditto for CSS and Javascript (up to a point). I'd _build_ things, instead of setting things up so I could finally get around to building something.

The first website I made was just HTML and CSS. The next one was Drupal. I didn't build on what I had learned with the first one, I jumped way ahead, and then wondered why I wasn't getting it. Having successfully ridden halfway down the block on training wheels, I proceeded to attempt driving a car. It didn't go well.

For a long time, I really thought I wasn't smart enough to learn web development. Which led to long periods of giving up. I thought, and this is important, that if I _didn't_ know how to do something, it was because I _couldn't_ know how to do it. Knowledge, and confidence, increase in small steps.

I was freed from this cycle eventually by way of a Brian Holt course on Frontend Masters. At one point he said, "Don't use Vim. Just use something easy." Or something like that. It was as if I had been given permission to not know how to do everything. It was ok if I didn't try to do the hardest thing, because it was ok if I didn't know yet how to do the thing that was just a little bit harder than what I already knew how to do.
